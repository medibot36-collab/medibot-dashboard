HELP & DOCUMENTATION

🔌 Connecting to Firebase

  • Open medbot_firebase.js and fill in your Firebase config values.
  • Go to Firebase Console → Project Settings → Your Apps.
  • Copy all 7 values into the firebaseConfig object.
  • The dashboard auto-connects on login — no IP needed.

📷 Camera Feed Setup

  • Use an ESP32-CAM module with the CameraWebServer sketch.
  • Ensure it's on the same Wi-Fi as your computer.
  • Enter its IP in the Camera Feed section and click Connect.
  • Use Speaker/Mic ON buttons to enable two-way audio.

❤️ Sensor Readings

  • MAX30102 sensor provides Heart Rate and SpO₂.
  • ESP32 writes data to medbot/vitals/P001/ in Firebase.
  • Z-score > 2 triggers Warning, > 3 triggers Danger.
  • Click Sound Alert Buzzer to activate robot buzzer.

🤖 Robot Control

  • Drag the joystick or use arrow keys to move.
  • Commands are written to Firebase in real-time.
  • Always press STOP before releasing the joystick.
  • Select a patient from the dropdown to navigate to their bed.
