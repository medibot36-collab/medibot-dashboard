HELP & DOCUMENTATION

🔌 Firebase Schema

  • Vitals: vitals/{patientId}/hr, spo2, timestamp
  • Patient: patients/{patientId}/name, age, gender, email
  • Reports: reports/{patientId}/pdfUrl
  • Active: Commands/activePatient

📄 PDF Report Flow

  • Backend generates PDF → uploads to Firebase Storage.
  • Writes URL to reports/{patientId}/pdfUrl.
  • Dashboard listener fires immediately — no refresh.
  • Click View Report to open latest PDF.

❤️ Sensor Readings

  • MAX30102 provides HR and SpO₂ data.
  • ESP32 writes to vitals/{patientId}/.
  • Z-score > 2 → Warning; threshold breach → Danger.
  • Risk meter color-codes severity in real time.

🤖 Robot Control

  • Drag joystick or use Arrow keys to move.
  • Enable Keyboard (WASD) for W/A/S/D + Space control.
  • Commands write to Commands/robot/ in real-time.
  • Path indicator shows current movement direction.
